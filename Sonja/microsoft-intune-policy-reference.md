# Référence de stratégie de configuration de Microsoft Intune

Utilisez les informations dans cette rubrique pour vous aider à décider quelle stratégie de configuration de Microsoft Intune vous devez utiliser pour gérer vos périphériques.

> [!TIP]
> Pour plus d’informations sur l’utilisation des stratégies, voir [Gérer les paramètres et fonctionnalités sur vos appareils avec Microsoft Intune stratégies](manage-settings-and-features-on-your-devices-with-microsoft-intune-policies.md).


## Stratégies de configuration Android

|Nom de la stratégie|Utilisez-la lorsque vous voulez|
|---------------|------------------------|
|**Configuration personnalisée (Android 4 et versions ultérieur, Samsung KNOX Standard 4.0 et versions ultérieur)**|Déployer les paramètres d’ouvrir Mobile Alliance Uniform Resource Identifier (URI OMA), tels que les paramètres Wi-Fi qui peuvent être utilisés pour contrôler les fonctionnalités de l’appareil. Cela est utile lorsque le paramètre dont vous avez besoin n’est pas disponible dans une stratégie de configuration.<br /><br />Pour plus d’informations, voir [paramètres de stratégie Android dans Microsoft Intune](android-policy-settings-in-microsoft-intune.md).|
|**Profil de messagerie (Samsung KNOX Standard 4.0 et versions ultérieur)**|Créer, déployer et surveiller les paramètres de messagerie Exchange ActiveSync sur les appareils gérées. Cela permet aux utilisateurs d’accéder à une messagerie d’entreprise sur leurs appareils personnels sans une configuration requise de leur part.<br /><br />Pour plus d’informations, voir [configurer l’accès à la messagerie d’entreprise en utilisant les profils de courrier électronique avec Microsoft Intune](configure-access-to-corporate-email-using-email-profiles-with-microsoft-intune.md).|
|**Configuration générale (Android 4 et versions ultérieur, Samsung KNOX Standard 4.0 et versions ultérieur)**|Configurer les paramètres fonctionnels et sécurité des périphériques mobiles.<br />Spécifier les applications qui sont conformes et rapports lorsqu’ils sont utilisés.<br />Configurer le mode plein écran que verrous appareils pour autoriser uniquement certaines fonctionnalités de fonctionner, par exemple, permettent à l’appareil exécuter une application ou de désactiver les boutons de volume.<br /><br />Pour plus d’informations, voir [paramètres de stratégie Android dans Microsoft Intune](android-policy-settings-in-microsoft-intune.md).|
|**PKCS #12 (. Profil de certificat PFX) (Android 4 et versions ultérieur)**|Utiliser ce profil pour créer et déployer. Paramètres de PFX des demandes de certificat d’appareil.<br /><br />Pour plus d’informations, voir [accès aux profils de certificat dans Microsoft Intune ressources sécurisée](secure-resource-access-with-certificate-profiles.md).|
|**Profil de certificat SCEP (Android 4 et versions ultérieur)**|Configurer un certificat Simple protocole d’inscription d’un certificat qui peut servir avec un certificat approuvé appareil mobile pour authentifier les périphériques mobiles pour lui permettre d’accéder aux ressources réseau tels que ceux qui sont configurés par Wi-Fi et profils VPN.<br /><br />Pour plus d’informations, voir [accès aux profils de certificat dans Microsoft Intune ressources sécurisé](secure-resource-access-with-certificate-profiles.md).|
|**Profil de certificat approuvé (Android 4 et versions ultérieur)**|Configurer un certificat approuvé appareil mobile qui peut être utilisé pour authentifier les périphériques mobiles pour lui permettre d’accéder aux ressources réseau tels que ceux qui sont configurés par Wi-Fi et profils VPN.<br /><br />Pour plus d’informations, voir [accès aux profils de certificat dans Microsoft Intune ressources sécurisée](secure-resource-access-with-certificate-profiles.md).|
|**Profil VPN (Android 4 et versions ultérieur)**|Configurer et déployer des paramètres qui offrent aux utilisateurs un accès sécurisé à votre réseau d’entreprise à partir de leur appareil mobile. En déployant ces paramètres, vous simplifiez les connexions aux utilisateurs finaux pour leur travail.<br /><br />Pour plus d’informations, voir [connexions VPN dans Microsoft Intune.md](vpn-connections-in-microsoft-intune.md).|
|**Profil de Wi-Fi (Android 4 et versions ultérieur)**|Configurer et déployer des paramètres réseau sans fil aux utilisateurs de votre organisation. En déployant ces paramètres, vous simplifiez connexions pour les utilisateurs finaux au réseau sans fil.<br /><br />Pour plus d’informations, voir [connexions Wi-Fi dans Microsoft Intune](wi-fi-connections-in-microsoft-intune.md).|
